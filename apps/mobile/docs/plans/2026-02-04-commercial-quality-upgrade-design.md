# MedNotiV2 상용 앱 고도화 설계

## 개요
- **타겟 사용자**: 비즈니스 사용자 (고객/거래처 메시지 분류 및 업무 추적)
- **수익 모델**: 추후 결정 (앱 품질 우선)
- **언어**: 한국어 전용
- **목표**: 현재 v1.0 기능 완성 앱을 상용 수준으로 고도화

---

## 섹션 1: 첫 인상 / 온보딩

### 스플래시 화면
- 앱 아이콘 + "MedNoti" 로고 페이드인
- 0.8~1초 표시 후 자연스럽게 전환
- 첫 실행 여부에 따라 온보딩 또는 메인 화면으로 분기

### 온보딩 플로우 (첫 실행 시 3단계)
1. **앱 소개** - "알림을 자동으로 캡처하고 업무로 관리하세요" + 핵심 기능 아이콘 설명
2. **권한 설정** - 알림 접근 권한, SMS 권한을 단계별로 안내 (PermissionHandler 통합)
3. **빠른 시작** - "첫 카테고리를 만들어보세요" 또는 "건너뛰기"

### 빈 상태 처리
- 메시지 없을 때: 아이콘 + "알림이 도착하면 여기에 표시됩니다" + 필터 설정 바로가기
- 카테고리 없을 때: "카테고리를 추가하면 메시지가 자동 분류됩니다" + 추가 버튼
- 각 화면별 맞춤 빈 상태 UI

---

## 섹션 2: 안정성 / 에러 처리

### 데이터베이스 안정화
- `fallbackToDestructiveMigration()` 제거 → 정식 마이그레이션 스키마
- 모든 DB 작업에 try-catch + 사용자 친화적 에러 메시지

### 권한 상태 관리
- 앱 재진입 시 권한 상태 재확인
- 권한 해제 시 부드러운 안내 배너 (크래시 대신)

### 입력 검증
- 카테고리/필터/상태 생성 시 빈 이름, 중복 이름 검증
- 최대 글자 수 제한

### 예외 상황 처리
- NotificationListenerService 연결 끊김 시 자동 재연결
- 로컬 우선 동작 보장

---

## 섹션 3: UI/UX 완성도

### 빈 상태 (Empty State)
- 각 화면별 맞춤 아이콘 + 안내 문구 + 액션 버튼

### 로딩 상태
- Shimmer/Skeleton 효과
- 액션 중 로딩 인디케이터

### 사용자 피드백
- Snackbar로 결과 알림 (저장, 삭제, 상태변경)
- 삭제 전 확인 다이얼로그
- 삭제 후 Undo 기능

### 애니메이션
- 화면 전환 슬라이드/페이드 트랜지션
- 목록 아이템 staggered 애니메이션
- 칸반 카드 이동 전환

### 스와이프 제스처
- 좌측 스와이프 → 삭제
- 우측 스와이프 → 다음 상태로 이동

### 기타 디테일
- 앱바 메시지 총 개수 배지
- 검색 하이라이트
- 긴 텍스트 말줄임 + 탭 확장

---

## 섹션 4: 상용 필수 기능 추가

### 데이터 백업/복원
- JSON 형식 전체 데이터 내보내기
- 파일에서 복원 (병합 또는 덮어쓰기 선택)
- 설정 > 일반 탭에 배치

### 통계 대시보드
- 메시지 탭 상단 요약 카드 (오늘 수신, 미처리, 완료)
- 캘린더 월간 통계 강화 (카테고리별 비율)

### 캡처 알림
- 새 메시지 캡처 시 상태바 알림
- 알림 탭 → 메시지 상세 이동
- 설정에서 ON/OFF 토글

### 일괄 작업
- 다중 선택 모드 (길게 눌러 진입)
- 일괄 상태 변경, 일괄 삭제
- 전체 선택/해제

### 메시지 관리
- 정렬 옵션 (최신순, 오래된순, 발신자순, 앱별)
- 자동 삭제 설정 (30/60/90일)
- 메시지 보관(아카이브) 기능
