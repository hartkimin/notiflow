# 웹 대시보드 스케줄 뷰 (2026-02-18)

## 요약

웹 대시보드의 캘린더 페이지를 모바일 앱의 스케줄 탭과 동일한 데이터/UI 구조로 전면 교체.
기존 월간 주문 캘린더 → 주간 카테고리별 플랜 관리 뷰로 전환.

## 주요 변경사항

### 새로운 스케줄 뷰
- **사이드바**: 미니 캘린더(주 선택), 카테고리 범례, 주간 작업 버튼
- **7컬럼 주간 그리드**: 월~일 각 날짜에 카테고리별 섹션 표시
- **주간 네비게이션**: `?week=yyyy-MM-dd` URL 파라미터, 이전/다음 주 이동, "오늘" 버튼

### 플랜 관리
- 카테고리별 인라인 플랜 추가
- 체크박스로 완료 상태 토글
- 주문번호 입력 (다이얼로그)
- 수신 메시지 연결/해제 (팝오버)
- 플랜 삭제 (소프트 삭제)

### 주간 일괄 작업
- **전주 불러오기**: 이전 주의 플랜과 카테고리를 현재 주로 복사
- **전체 카테고리 추가**: 모든 활성 카테고리를 7일 전체에 추가
- **다음주로 복사**: 현재 주의 플랜과 카테고리를 다음 주로 복사

### 메시지 확인
- 카테고리별 수신 메시지 수 배지
- 팝오버로 메시지 내용 미리보기
- 플랜에 메시지 연결 시 링크 아이콘과 툴팁 표시

## 데이터 레이어

모바일 앱과 동일한 Supabase 테이블 사용:
- `categories` - 카테고리 목록 (ARGB 색상)
- `plans` - 플랜 항목
- `day_categories` - 날짜별 카테고리 할당
- `captured_messages` - 모바일 수신 메시지
